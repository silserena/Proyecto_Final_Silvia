Ecom web

Esto es un prototipo de web ecommerce hecho en Figma, con la intención de aprender a hacer diferentes interacciones con un userflow.

Para aprender a hacerlo tome de referencia diferentes paginas web, entre ellas Zara. A partir de ahi fui creando una pequeña galeria, y me basé en un flowchart.

El user tenia que entrar en camisas de mujer, y elegir dos de diferenets tallas, podia interactuar con la web también eliminando el artículo.

Previo al diseño definitivo, trabajé con wireframes para saber cómo queria que fuese la web.
